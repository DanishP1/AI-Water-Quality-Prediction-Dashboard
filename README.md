# AI-Powered Water Quality Prediction Dashboard

This project is an AI-based water quality prediction and analytics system developed using Machine Learning, MySQL, and Tableau.

## Technologies Used
- Python
- Machine Learning
- Random Forest
- MySQL
- Tableau
- Pandas
- Scikit-learn

## Features
- Water quality prediction
- Safe/Unsafe classification
- MySQL database integration
- Tableau dashboard analytics
- Feature importance analysis
